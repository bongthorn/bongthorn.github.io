## Add Prettier plugin

:::info
By adding this plugin, editor will automaticaly format your code pretty.
:::

### Install

https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode

### Setting

View -> Command Palette -> search: "open user settings" -> search: "format on save" -> check
![format-on-save-vscode](https://storage.googleapis.com/coderhackers-assets/the-complete-webdev-with-rails-2020/vscode-format-on-save.gif)

## Change tabsize

View -> Command Palette -> search: "open user settings" -> search: "tabsize"
![tab-setting](https://storage.googleapis.com/coderhackers-assets/the-complete-webdev-with-rails-2020/tab-indent-vscode-setting.png)

Confirm **`Tab Size`** is **"2"**.
