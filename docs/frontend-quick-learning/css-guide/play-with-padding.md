## Padding - Shorthand Property

```css
/* Apply to all four sides */
padding: 10px;

/* vertical | horizontal */
padding: 5px 10px;

/* top | horizontal | bottom */
padding: 1px 2px 2px;

/* top | right | bottom | left */
padding: 5px 1px 0 2px;
```
