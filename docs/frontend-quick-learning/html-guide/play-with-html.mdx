## Let's try some new HTML tag

We start from this code.

```html title="test1.html"
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    Hello World!
  </body>
</html>
```

Okay, so let's add new HTML tags.

### `<h1>` and `<h2>` tag

![w3-school-h1-h6.png](https://storage.googleapis.com/coderhackers-assets/the-complete-webdev-with-rails-2020/html-guide/w3-school-h1-h6.png)
https://www.w3schools.com/tags/tag_hn.asp

Let's write manually and change the content of h1 and h2.

```html title="test1.html"
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <!-- highlight-start -->
    <h1>Here is a title.</h1>
    <h2>Here is a subtitle.</h2>
    <!-- highlight-end -->
  </body>
</html>
```

Check the output.

![h1-h2-output.png](https://storage.googleapis.com/coderhackers-assets/docs/img/2020-04-27-03-20-34.png)

### `<ul>` tag

https://www.w3schools.com/tags/tag_ul.asp

```html title="test1.html"
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <h1>Here is a title.</h1>
    <h2>Here is a subtitle.</h2>

    <!-- highlight-start -->
    <ul>
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ul>
    <!-- highlight-end -->
  </body>
</html>
```

#### Check the output.
